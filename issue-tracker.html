Project Overview

A modern, full-featured issue tracking application built with the MERN stack (MongoDB, Express, React, Node.js) + TypeScript. Designed to help teams efficiently create, assign, track, and resolve bugs and tasks.

Key Features

User Authentication

Secure login/signup with role-based access (Admin & User)

When logging in access is granted based on user role

Issue Management

Create, edit, assign, and update issues with status tracking

Comment System

Real-time threaded comments on every issue

Advanced Filtering

Filter by status, priority, assignee, and search functionality

Live Deployment

Deployed on Google Cloud with full backend API

Responsive Dashboard

Clean, intuitive UI that works on all devices
