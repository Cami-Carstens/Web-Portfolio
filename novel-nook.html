Novel Nook

Full-Stack ASP.NET MVC Bookstore Application

Watch Demo

Project Overview

A comprehensive online bookstore and lifestyle shop built with ASP.NET MVC and SQL Server. Features a full shopping experience including books, clearance items, home decor, and community book reviews.

Key Features

Book Catalog

Browse and search books by title, author, or genre

Clearance Section

Discounted books and special deals

Home Decor Catalog

Lifestyle and home decor products

Book Talk Reviews

Community book reviews and discussion section

Shopping Cart

Add items from any section and checkout

User Authentication

Registration, login, and role-based access

Admin Panel Features

Full inventory management across all sections:

📚 Books
🏷️ Clearance
🏠 Home Decor
💬 Book Reviews

Admins can add, edit, delete, and manage inventory for every category.

What I Learned

  • MVC architectural pattern in a real-world e-commerce app
  • Entity Framework Core with complex database relationships
  • Building multiple catalog sections with shared shopping cart
  • Role-based authorization (Customer vs Admin)
  • Implementing inventory management across different product types
  • Form validation, sessions, and state management
  • Creating user-friendly admin dashboards
  • SQL Server design for multi-category e-commerce
